# Openclassroom
Mini-jeu en JavaScript développé dans le cadre d'un cours OpenClassRoom.

Nécessite d'être ouvert sur un serveur local virtuel.


Règles : 
	-2 joueurs au tour par tour.
	-les joueurs peuvent ramasser des armes sur la carte afin d'augmenter leurs dégâts.
	-si les deux joueurs se rencontrent, un combat à mort commence.
